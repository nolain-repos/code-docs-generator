# mkdocs-setup Documentation

## Description of the project

**mkdocs-setup** is a command-line tool designed to streamline the creation and management of professional documentation for Python projects. It automates the generation of technical reference guides, integrates AI-assisted documentation writing, and provides an interactive setup wizard to simplify the documentation process.

Key features include:
- Automatic generation of API documentation from Python module docstrings
- AI-powered content creation for project documentation
- Interactive configuration wizard for easy setup
- Integration with MkDocs for static site generation
- Support for multiple LLM providers (OpenAI, Google, DeepSeek, Mistral)
- Environment-based API key management
